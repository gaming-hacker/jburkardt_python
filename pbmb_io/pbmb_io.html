pbmb_io

pbmb_io, a Python code which reads and writes files in the binary Portable Bit Map (PBM) format for 2D bit map graphics information.

There are many graphics programs, such as XV and GIMP, that can display images in this format, or translate them to other formats such as BMP, GIF, JPEG, MacPaint, X Window, and so on.

Related Data and Programs:

pbma_io, a Python code which handles the ASCII Portable Bit Map (PBM) format.

pbmb, a data directory which contains some example binary Portable Bit Map (PBM) files.

pgmb_io, a Python code which handles the binary Portable Gray Map (PGM) format.

ppmb_io, a Python code which handles the binary Portable Pixel Map (PPM) format.
