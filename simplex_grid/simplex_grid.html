simplex_grid

simplex_grid, a Python code which generates a grid of points over the interior of a simplex in M dimensions.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

simplex_grid is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
