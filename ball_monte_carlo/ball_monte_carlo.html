ball_monte_carlo

ball_monte_carlo, a Python code which estimates the integral of F(X,Y,Z) over the interior of the unit ball in 3D.

Licensing:

The computer code and data files made available on this web page are distributed under the GNU LGPL license.

Languages:

ball_monte_carlo is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
